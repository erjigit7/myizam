# Полевые заметки: чем реальность отличается от спецификации

Найдено при интеграции 15.07.2026. Каждый пункт подтверждён на реальных данных;
почти на каждый есть юнит-тест с фикстурой.

## API

1. **WAF блокирует нестандартный User-Agent.** «Вежливый» `Myizam-Ingestion/1.0 (+github…)`
   получает `403 Forbidden` (тело: `Forbidden, Transaction ID: …`). Работает только
   браузерный UA. Спецификация §5 рекомендовала обратное — не следовать ей.
2. **Rate limit: ~20 запросов за 5 секунд** (общий на IP). При превышении — HTTP 200
   с ПЛОСКИМ ТЕКСТОМ `API calls quota exceeded! maximum admitted 20 per 5s.` вместо JSON.
   Клиент ретраит любой не-JSON ответ с бэкоффом.
3. **Даты редакций в `editions[].nameRus` — зоопарк:** `23.06.2026`, `14.11.2025 № 257`,
   `20/06/2005`, `21,06,2018`, `02 .12.2024`. Regex: `^(\d{1,2})\s*[./,]\s*(\d{1,2})\s*[./,]\s*(\d{4})`.
4. `GetDocuments` (поиск) — POST с телом `{"nameRus": "...", "isPercentage": true}`,
   ответ `{data: [...], totalResultsCount}`. В выдаче есть «Утратил силу» — фильтровать по status.

## Word-HTML

5. **`<meta charset=windows-1251>` внутри contentRu.** Строка уже декодирована из JSON,
   но AngleSharp честно перекодирует по meta и превращает текст в кашу. Вырезать meta до парсинга.
6. **Вставные статьи — тремя способами:**
   - `<sup>` элемент: `Статья 21<sup>1</sup>.` (УК) → 21-1; точка бывает ВНУТРИ:
     `Статья 317<sup>1.</sup>` (КоП);
   - юникодная сущность: `Статья 240&sup1;.` = `240¹` (УК) → 240-1;
   - без обработки «211» сталкивается с настоящей статьёй 211 УК («Угон…»).
   Замена только в контексте «после цифры» — чтобы не трогать «м²» в таблицах штрафов.
7. **Кириллица вместо цифр в номерах:** «Статья З09-3» с буквой «З» (КоП 3-36).
   Правим конфузаблы (З→3, О→0) только в токене номера после «Статья».
8. **Заголовки статей всегда с точкой после номера** («Статья 82. Название»), а вот
   упоминание в начале абзаца — без («Статья 21 настоящего Кодекса не применяется…»).
   Поэтому у статьи точка после номера обязательна. У Глав/Разделов — нет:
   в ГК-I «Глава 10-1 Право собственности…» идёт без точки.
9. **Нумерация статей НЕ монотонна.** В ГК-I вставная Глава 10-1 (статьи 233-1…233-19)
   физически стоит после статьи 244. Проверка порядка — предупреждение, не ошибка.
10. **Дубли целых статей в источнике:** статья 386 НК присутствует дважды подряд
    (полный текст оба раза). Парсер схлопывает: побеждает последняя копия.
11. **Якоря `<a name=st_N>` неполные и с опечатками.** В ТК-2025 покрытие 100%
    (265/265), в ГК-I — 80 якорей на 440 статей. Опечатки: st_126 указывает на
    статью 726 (ГК-II), st_563_1 — на статью 563 (КоП). Якоря годятся только
    для предупреждений, не как эталон.
12. **Большие дыры в нумерации — норма:** в СК статьи 126–162 исключены целым блоком
    (их нет в HTML), в НК нет статей 387, 395, 403–408. Подтверждено по источнику.
13. Разделы в новых кодексах — ЗАГЛАВНЫМИ («РАЗДЕЛ I. ОБЩИЕ ПОЛОЖЕНИЯ»), в УК/КоП
    название раздела идёт отдельным абзацем после «РАЗДЕЛ VII».
